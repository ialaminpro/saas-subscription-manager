== Description ==
SaaS Subscription Mmanager is a WordPress plugin that helps you manage your SaaS platform subscriptions, analytics, and automate cron jobs. It also includes REST API integration for fetching data externally.

== Installation ==
1. Upload the `saas-subscription-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the new "Subscriptions" and "Analytics" pages in the WordPress admin menu.
